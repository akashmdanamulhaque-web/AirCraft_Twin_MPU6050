# AirCraft Twin MPU6050

A real-time aircraft attitude visualization system using an MPU6050 sensor, ESP32, Python, and Panda3D.

## Features

- Real-time Roll, Pitch and Yaw tracking
- 3D aircraft visualization
- Live sensor feedback
- Serial communication via ESP32
- Digital Twin representation of aircraft orientation

## Hardware

- ESP32 DevKit V1
- MPU6050

## Software

- Python
- Panda3D
- PySerial

## Files

- `dashboard.py` - Flight dashboard visualization
- `mpu_board.py` - MPU6050 orientation visualization
- `Jet.glb` - 3D aircraft model
